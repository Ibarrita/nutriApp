# Aplicación nutricional
## Introducción
### Objetivos de la investigación
Se llevó a cabo una investigación de 3 aplicaciones nutricionales tomando en cuenta sus funcionalidades principales, la experiencia que vive el usuario al usarse, si tienen algún costo, los puntos débiles, fuertes y sus características, todo esto fue con el propósito de conocer cómo funcionan y que nos ofrecen para poder obtener ideas. Con toda esta información recaudada nuestro objetivo principal y la razón por la cual estamos haciendo todo este proceso es diseñar una aplicación sencilla pero que sea útil, para ayudar a las personas a comer de una manera más saludable, decidimos enfocarnos en la alimentación y nutrición porque México ocupa actualmente el primer lugar mundial en obesidad lo cual es un problema serio, para poder saber nosotros cual era la mejor manera de llegar al público y que esta propuesta fuera realmente útil se llevó a cabo una encuesta para conocer la opinión y las necesidades de los usuarios, de esta manera detectamos los problemas más comunes y conocemos que funciones, diseños y herramientas son más útiles y atractivas para el público.

### Metodología utilizada
Para realizar esta investigación se llevó a cabo un proceso fácil y sencillo para poder conocer mejor cómo funcionan las aplicaciones nutricionales.

Primero seleccionamos tres aplicaciones nutricionales diferente, tomamos en cuenta que estas estuvieran disponibles en cualquier dispositivo, se instalaron e investigamos las funciones, el diseño y las herramientas que cada aplicación tiene, después de ver esto reflexionamos que aspectos se podían cambiar y mejorarse. Esto ayuda para darnos una idea de cómo son estas aplicaciones, que tan fáciles son de usar y que tipo de ayudan le brindan al público.

Después en base a la información que ya sabíamos sobre aplicaciones realizamos una encuesta que era para todo tipo de público. En esta encuesta se incluyeron preguntas de respuesta abierta y cerrada sobre los hábitos alimenticios, que cambiarían de estos hábitos, las experiencias personales al usar aplicaciones de este tipo, los problemas más comunes, que funciones que podemos brindar les resulta más útiles y que tipo de diseño prefieren.

Toda esta información se organizó en gráficas para identificar de manera más fácil las respuestas que más se repetían y que buscaba principalmente el público en una aplicación nutricional.

Para finalizar, basándonos en los resultados de la encuesta elaboramos una propuesta de diseño para una aplicación enfocada en ayudar a tener hábitos y una alimentación más saludable y adaptada a las necesidades de cada persona.

## Análisis de aplicaciones
### Análisis individual de cada aplicación
#### MyFitnessPal
Al parecer es la más popular, es gratuita, aunque tiene un plan de pago, además de otras características como que se puede conectar con otras aplicaciones, una base de datos completa y también se puede usar en relojes inteligentes.

- Funcionalidades principales: Contador de calorías, base de datos y registro de alimentos.
- Experiencia de usuario: Facilidad de uso y diseño amigable.
- Modelo de negocio: Freemium.
- Puntos fuertes: Facilidad de uso, se puede vincular la cuenta de MyFitnessPal con otras aplicaciones y se puede tener un registro diario de alimentos.
- Puntos débiles: No lo pueden usar menores de 18 años según sus términos y condiciones, por lo que no todos pueden tener la oportunidad de usarla.
- Características innovadoras: Se conecta con otras aplicaciones y también tiene foros.

#### FatSecret
Su objetivo principal es la pérdida de peso, su base de datos es algo completa, aunque tiene mucho menos que la de MyFitnessPal, además que la API de esta aplicación está disponible para usarse por otras personas.

- Funcionalidades principales: Contador de alimentos, base de datos, registro de alimentos y escáner de código de barras.
- Experiencia de usuario: Es fácil de usar y tiene un diseño intuitivo.
- Modelo de negocio: Freemium.
- Puntos fuertes: Facilidad de uso, es una aplicación completa y tiene 1.9 millones de alimentos en su base de datos.
- Puntos débiles: No se integra con otras aplicaciones.
- Características innovadoras: Utiliza su propia API.

#### Yazio
Podría ser la opción menos popular, su objetivo es la vida sana en general, se puede conectar con otras aplicaciones y además tiene varios planes de ayuno, como 16/8, 5/2 y 6/1.

- Funcionalidades principales: Contador de calorías, cuenta con diferentes calculadoras cómo una calculadora de IMC, base de datos y recetas.
- Experiencia de usuario: Es fácil de usar y tiene un diseño intuitivo.
- Modelo de negocio: Freemium.
- Puntos fuertes: Se puede conectar a una pulsera de actividad para tener un registro automático y también se puede usar como una aplicación web.
- Puntos débiles: No tiene muchas funciones.
- Características innovadoras: Se puede conectar a una pulsera de actividad y también tiene planes de ayuno intermitente.

## Diseño y resultado
### Diseño: Justificacion de preguntas y metodologia
La encuesta fue diseñada con el propósito de conocer mejor a las personas interesadas en mejorar su alimentación y su experiencia con las aplicaciones de nutrición.

Las preguntas se elaboraron en un lenguaje claro y fácil de entender para que cualquier persona pudiera responder, aun si nunca ha usado una aplicación de este tipo.

Se incluyeron preguntas **cerradas** (con opciones de respuesta) para obtener datos concretos, y **preguntas abiertas** para conocer opiniones, sugerencias y experiencias personales.
Los temas principales de la encuesta fueron:
- Perfil del usuario (edad, intereses y hábitos).
- Experiencia o conocimiento previo sobre aplicaciones de nutrición.
- Funciones que consideran más útiles e innovadoras.
- Problemas comunes al usar este tipo de apps.
- Preferencias en diseño, colores y estilo visual.
- Ideas o sugerencias para mejorar una app de nutrición.

La encuesta se aplicó de manera digital, utilizando Google Forms, para facilitar la participación de diferentes personas y recolectar los datos de forma rápida y ordenada.

### Hallazgos principales: Lo que los usuarios realmente buscan
Del análisis de las respuestas se obtuvieron varias conclusiones importantes:

- Los usuarios quieren **apps simples, intuitivas y personalizadas**, que se adapten a su estilo de vida.
- Les interesa **recibir consejos útiles y motivación**, no solo contar calorías.
- Prefieren una aplicación que **no dependa de pagos constantes** y que ofrezca **contenido accesible y confiable**.
- Las personas valoran las funciones que **ahorran tiempo**, como planificadores automáticos o recordatorios inteligentes.
- La mayoría desea que la app **se adapte a sus necesidades personales**, por ejemplo, si son vegetarianos, hacen ayuno intermitente o tienen alguna alergia.

## Conclusiones y Oportunidades
### Síntesis de aprendizaje
Síntesis de aprendizajes: Existe una gran variedad de aplicaciones de salud y fitness, especialmente las de nutrición, las cuales ofrecen un contador de calorías, una base de datos de alimentos, recetas saludables, etc

### Oportunidades identificadas para el desarrollo futuro
Que ayude a las personas a tener hábitos saludables, tales como bajar de peso, mantener el peso, dieta balanceada, ganar músculo, etc.

### Recomendaciones para la futura aplicación
Un diseño amigable, que sea fácil de usar y que ofrezca solo lo necesario.
